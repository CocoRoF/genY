# Progress Index

실제 구현이 시작되면 PR 단위로 기록한다. 아직 사용자 검토/승인 이전이므로
본 폴더는 비어있다 — `README.md` 에 작업 리듬과 기록 포맷만 명시한다.

## 문서

- [README.md](README.md) — 기록 규약 (파일명/필드/PR 번호 매핑).

## 예정 구조

```
progress/
├── README.md
├── index.md
└── NN_<short-slug>.md   # 각 PR 마다 한 파일
```

각 진행 문서의 섹션:
- **Scope** — 다룬 범위
- **PR Link** — 머지된 PR URL
- **Summary** — 변경 요약
- **Follow-ups** — 차후 PR 로 미룬 항목
